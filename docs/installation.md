# Installation

## From PyPI

```bash
pip install pynebulosa
```

## With scanpy (recommended)

```bash
pip install "pynebulosa[scanpy]"
```

## Development install

```bash
git clone https://github.com/mdmanurung/pyNebulosa.git
cd pyNebulosa
pip install -e ".[dev]"
```

## Requirements

- Python >= 3.9
- numpy >= 1.22
- scipy >= 1.9
- matplotlib >= 3.5
- anndata >= 0.8
- scanpy >= 1.9 (optional, for loading example datasets)
